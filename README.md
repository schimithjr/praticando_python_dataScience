# Projeto Semana 0 - Simulador de Vendas

Este projeto foi desenvolvido durante minha semana de ambientação em Python.
Possui como intenção finalizar a primeira etapa de aprendizado, que 
nomeei como "Ambientação"
O script simula um sistema de checkout que processa múltiplas vendas, 
calcula descontos progressivos e armazena o histórico em dicionários.
O objetivo de desenvolver esse script foi para praticar e colocar em prova
conceitos básicos estudados.

### Tecnologias Utilizadas:
- Python 3
- Estruturas de Dados (Listas, Tuplas, Dicionários)
- Tratamento de Erros (Try/Except)
- Laços de Repetição (While/For)
